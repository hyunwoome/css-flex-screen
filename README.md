## 🍌 바닐라 자바스크립트를 사용한 CSS 치트 시트

- HTML5
- CSS
- JavaScript

<br>

## ⚙️ 로컬환경에서 실행

- git clone

```sh
$ git clone https://github.com/hyunwoome/css-live-screen.git
```

- 패키지 설치

```sh
$ cd css-live-screen
$ npm install
```

- vscode liveserver로 개발서버 실행
